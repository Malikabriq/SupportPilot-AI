# SupportPilot AI

An AI-driven support ticket management system.

## Structure
- `backend/`: FastAPI application with AI services.
- `frontend/`: Next.js dashboard.
- `scripts/`: Utility scripts.

## Setup
1. Backend:
   ```
   cd backend
   pip install -r requirements.txt
   uvicorn app.main:app --reload
   ```
2. Frontend:
   ```
   cd frontend
   npm install
   npm run dev
   ```
